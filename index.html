Mengenal Sanma Gallery

Sanma Gallery adalah platform portfolio digital yang menggabungkan estetika visual dengan teknologi web modern. Website ini dibuat untuk mendokumentasikan perjalanan kreatif saya dalam dunia desain dan programming.

Sanma Gallery adalah sebuah website untuk menyimpan foto kenangan agar bisa berbagi momen lama yamg telah di alami bersama kawan tanpa perlu mengirim 1 per 1 foto di whatsApp, hanya perlu buka situs dan kamu bisa melihat semuanya di sini.

website ini juga bisa diperbarui jika ada dukukung dari kawan-kawan sekalian, lalu jika ada kritik dan sarannya atau jika menemukan bug bisa di sampaikan melaluiContact ini

  • ✅ Manajemen 180+ foto secara dinamis
  • ✅ Antarmuka responsif dengan Tailwind CSS
  • ✅ Pengalaman user yang interaktif
